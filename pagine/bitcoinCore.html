Avvio nodo

per avviare il demone:

$ bitcoind

le impostazioni verranno prese dal file di configurazione presente in $USER/.bitcoin/bitcoin.conf. Le eventuali impostazioni inserite da riga di comando hano la precedenza sul file di configurazione.
Per stoppare il demone:

$ bitcoin-cli stop
$ bitcoin-cli getblockchaininfo
$ bitcoin-cli getrawmempool
